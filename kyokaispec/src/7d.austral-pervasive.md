## Austral.Pervasive Module {#austral.pervasive}

The `Austral.Pervasive` module exports declarations which are imported into
every module.

### Core Types and Entrypoint Support {#austral.pervasive-core}

```austral
union Option[T: Type]: Type is
    case None;
    case Some is
        value: T;
end;

union Either[L: Type, R: Type]: Type is
    case Left is
        left: L;
    case Right is
        right: R;
end;

generic [T: Type, R: Region]
function spanLength(sp: Span[T, R]): Index;

generic [T: Type, R: Region]
function spanWriteLength(sp: Span![T, R]): Index;

function abort(message: Span[Nat8, Static]): Unit;

type RootCapability : Linear;
function surrenderRoot(cap: RootCapability): Unit;

function argumentCount(): Index;
function nthArgument(n: Index): Span[Nat8, Static];

union ExitCode: Free is
    case ExitSuccess;
    case ExitFailure;
end;
```

`Option` and `Either` are pervasive sum types.

`abort` terminates the program after printing the given message.

`argumentCount` and `nthArgument` expose command-line arguments captured by the
generated C entrypoint.

### Arithmetic and Bitwise Typeclasses {#austral.pervasive-arithmetic}

```austral
typeclass TrappingArithmetic(T: Type) is
    method trappingAdd(lhs: T, rhs: T): T;
    method trappingSubtract(lhs: T, rhs: T): T;
    method trappingMultiply(lhs: T, rhs: T): T;
    method trappingDivide(lhs: T, rhs: T): T;
end;

typeclass ModularArithmetic(T: Type) is
    method modularAdd(lhs: T, rhs: T): T;
    method modularSubtract(lhs: T, rhs: T): T;
    method modularMultiply(lhs: T, rhs: T): T;
    method modularDivide(lhs: T, rhs: T): T;
end;

typeclass BitwiseOperations(T: Type) is
    method bitwiseAnd(lhs: T, rhs: T): T;
    method bitwiseOr(lhs: T, rhs: T): T;
    method bitwiseXor(lhs: T, rhs: T): T;
    method bitwiseNot(value: T): T;
end;

typeclass Remainder(T: Type) is
    method rem(lhs: T, rhs: T): T;
end;
```

Built-in `TrappingArithmetic`, `ModularArithmetic`, and `BitwiseOperations`
instances exist for the integer types, `Index`, and `ByteSize`, with
`TrappingArithmetic(Float64)` also provided. `Remainder` instances exist for the
unsigned integer types, signed integer types, and `Index` in the current
interface. The current `Pervasive.aum` body also contains an additional
duplicated `instance Remainder(Nat8)` block whose method signature is for
`Int8`; treat the interface as the authoritative exported surface and the body
duplicate as a compiler-source defect.

### Printing {#austral.pervasive-printing}

```austral
typeclass Printable(A: Free) is
    method print(value: A): Unit;

    generic [R: Region]
    method printSpan(value: Span[A, R]): Unit;
end;

generic [T: Free(Printable)]
function printLn(value: T): Unit;
```

Built-in `Printable` instances exist for `Unit`, `Bool`, all integer types,
`Index`, `ByteSize`, `Float32`, `Float64`, and spans of printable free values.

### Numeric Conversion Families {#austral.pervasive-conversions}

The module defines checked conversion typeclass families `ToNat8`, `ToNat16`,
`ToNat32`, `ToNat64`, `ToInt8`, `ToInt16`, `ToInt32`, `ToInt64`, `ToIndex`,
`ToFloat32`, and `ToFloat64`.

Each family has a single conversion method named after the target type, and each
family has built-in instances across the numeric built-in types supported by
`Austral.Pervasive`.

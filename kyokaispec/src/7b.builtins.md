# Built-In Modules {#builtin}

The compiler wires in a small standard library. The authoritative interfaces and
bodies live under `austral/lib/builtin/` in the reference compiler tree:

| Module | Interface | Body | Also referenced from |
| ------ | --------- | ---- | --------------------- |
| `Austral.Pervasive` | `Pervasive.aui` | `Pervasive.aum` | `BuiltIn.ml` (`pervasive_imports`) |
| `Austral.Memory` | `Memory.aui` | `Memory.aum` | `BuiltIn.ml` (`memory_module_name`, pointer/address helpers) |

Imports from `Austral.Pervasive` are injected into every compilation unit; see
`BuiltIn.pervasive_imports` for the exact symbol list. The compiler injects
these imports during `Compiler.parse_and_combine` for every module except
`Austral.Pervasive` itself.

Details of `Austral.Pervasive` appear in [Austral.Pervasive](7d.austral-pervasive.md);
`Austral.Memory` is documented in [Austral.Memory](7c.austral-memory.md).
